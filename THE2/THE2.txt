In this Take-Home Exam (THE), you will be practicing the concept of 2D Linked Lists. You
will implement a maze game using a 2D linked list structure. The tasks include reading the content of a
file that represents the maze, constructing a 2D Linked List and then populating the content
of the file to the 2D linked List.
